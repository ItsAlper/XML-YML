# Filmy

index.xml

### `<filmy>`
### `<film>`
- Atribut pro Film
- Podkategorie `<filmy>`
- Atributy:
    - `id`: identifikátor
    - `name`: název filmu
    - `year`: rok vydání

### `<Zanry>`
- **Hlavní atribut** 
- Atribut pro žánry
- Podkategorie `<filmy>`

### `<zanr>`
- Atribut pro žánr
- Podkategorie `<Zanry>`
- Atributy:
    - `id`: identifikátor
    - `name`: název žánr
